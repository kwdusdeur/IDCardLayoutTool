# 证卡裁剪打印工具 - .NET 8 版

Windows 11 桌面应用，用于自动裁剪和拼版证件照片。

## 功能特性

✅ **AI 自动裁剪**
- 三层算法：增强边缘检测 + 自适应矩形 + Canny凸包
- 自动旋转修正（横拍/竖拍都能识别）
- 支持：身份证、银行卡、驾驶证、护照

✅ **智能界面**
- 三栏布局：图片列表 + 预览区 + 工具栏
- 拖拽添加图片
- 缩略图显示
- 实时预览

✅ **色彩调整**
- 亮度调整：-100 ~ +100
- 对比度调整：-100 ~ +100
- 饱和度调整：-100 ~ +100

✅ **A4 拼版**
- 正反面拼版
- 间隔 55mm（精确）
- 300 DPI 输出

✅ **导出打印**
- 导出 PDF
- 导出图片（JPG/PNG）
- 直接打印

## 技术栈

- **框架**: C# + .NET 8 + WPF
- **图像处理**: Emgu.CV (OpenCV .NET 封装)
- **PDF**: PdfSharp
- **打包**: 单文件 exe，自包含运行时

## 编译

需要安装 .NET 8 SDK：https://dotnet.microsoft.com/download

```bash
# 方法1：使用脚本（推荐）
build.bat

# 方法2：手动命令
dotnet restore
dotnet publish -c Release -r win-x64 --self-contained true -p:PublishSingleFile=true -o publish
```

生成的 exe 在 `publish/` 目录。

## 使用方法

1. 双击 `证卡裁剪打印工具.exe`
2. 拖拽图片到窗口，或点击"添加图片"
3. 点击"一键裁剪"
4. 调整色彩（可选）
5. 导出或打印

## 核心算法

### 自动裁剪（三层保险）

1. **增强边缘检测**
   - CLAHE 对比度增强
   - 自适应 Canny 阈值
   - 轮廓四边形近似
   - 透视变换

2. **自适应矩形检测**
   - 自适应二值化
   - 形态学闭运算
   - 最小外接矩形
   - 透视变换

3. **Canny + 凸包**
   - Canny 边缘检测
   - 凸包简化
   - 四边形近似
   - 透视变换

### 自动旋转修正 🔥

所有方法裁剪后自动检测：
- 如果 `宽度 < 高度` → 自动旋转 90°
- 确保身份证/银行卡等横向卡片正确方向

### 保底策略

如果三层算法都失败：
1. Otsu 二值化找非白区域
2. 裁掉 5% 边距
3. 最后保底：裁掉 10% 边距
